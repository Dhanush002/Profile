Flower Shop Android Application

Description: The Flower Shop Android application is designed for a local flower shop named "X." The application allows users to interact with the flower shop online, providing features such as user authentication, product viewing, and online purchasing. Below is a detailed overview of the project, including the technologies used, concepts implemented, and a link to view the source code.

Tech Stacks Used:

1. Java: The application is developed using Java, a versatile and widely used programming language for Android app development.

2. Firebase Authentication: Firebase authentication is used for secure user authentication, enabling users to register and log in to their accounts.

3. Firebase Realtime Database: Firebase Realtime Database is employed to store and retrieve data related to flowers, user details, and orders.

4. Token Handling: Token-based authentication is implemented for enhanced security, ensuring secure communication between the client and the server.

5. UI/UX Design Concepts: The application incorporates UI/UX design principles to provide an intuitive and visually appealing user interface. This includes considerations for navigation, user feedback, and overall user experience.
